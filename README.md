# Xiaoyu Xu - Personal Academic Homepage

Personal academic website for Xiaoyu Xu, an undergraduate student at Fuzhou University researching LLM Interpretability, Multimodal Learning, and RAG Systems.

## Live Website

Visit: [https://xiaoyuxu-vincent.github.io](https://xiaoyuxu-vincent.github.io)

## Features

- Clean, minimalist academic design
- Fully responsive (desktop, tablet, mobile)
- Smooth scrolling navigation
- Publication showcase with links
- Research experience timeline
- Skills and awards sections

## Project Structure

```
├── index.html          # Main HTML file
├── css/
│   └── style.css       # Stylesheet
├── js/
│   └── main.js         # JavaScript interactions
├── images/
│   ├── avatar.jpg      # Profile photo (add your own)
│   └── fzu-logo.png    # University logo (optional)
└── README.md           # This file
```

## Setup

1. Add your profile photo as `images/avatar.jpg`
2. (Optional) Add Fuzhou University logo as `images/fzu-logo.png`
3. Update Google Scholar link in `index.html`
4. (Optional) Add CV PDF and update the download link

## Customization

- **Colors**: Edit CSS variables in `css/style.css`
- **Content**: Edit sections in `index.html`
- **Links**: Update social links in the hero section
